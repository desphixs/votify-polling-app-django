# Build Votify: A Live Community Polling App

### Project Description

You are going to build Votify a real time polling application where users can vote on burning questions and instantly see the results. First you will build a complex database where one single poll contains multiple different choices. Then you will write the logic to display these polls and handle the votes. But here is the trick: instead of just saving a new vote you will actually grab the current vote count and increment it. Finally you will write some Python math to calculate exactly what percentage of the crowd voted for each option and use those numbers to build dynamic visual progress bars in your HTML. You will do all of this using function based views to master exactly how data is processed.
